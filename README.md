# hr-attrition-decision-intelligence

# Project Overview
This project delivers an enterprise-level HR attrition analytics and decision intelligence system built using Power BI.

It transforms workforce data into strategic insights, enabling leadership to:
 - Identify high-risk employees
 - Analyze attrition drivers
 - Simulate salary hike impact scenarios
 - Support data-driven HR decision-making
 
# Business Problem 

Employee attrition increases hiring costs, disrupts operations, and impacts organizational stability.

This system answers:
 1. What factors drive attrition?
 2. Which departments are at risk?
 3. How does business travel impact churn?
 4. Can salary adjustments reduce high-risk attrition?
